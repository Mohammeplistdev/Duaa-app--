### طريقة التثبيت / Installation:
1. حمل ملف `duaa.ipa` من قسم الـ Releases.
   Download the `duaa.ipa` file from the Releases section.
2. تأكد من تثبيت **AppSync Unified** من السيديا.
   Make sure **AppSync Unified** is installed from Cydia.
3. استخدم **IPA Installer** لتثبيت التطبيق مباشرة من الجهاز.
   Use **IPA Installer** to install the app directly on your device.
4. يمكن التثبيت أيضاً عبر الكمبيوتر باستخدام **3uTools**.
   Can also be installed via PC using **3uTools**.
